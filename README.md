# 🎭 Face Morph Detection System

## 📌 Overview
This project is a **Face Morph Detection System** that identifies whether a given video is **Original** or **Morphed** using a combination of **Deep Learning and Machine Learning techniques**.

The system processes video frames, extracts facial features using a pre-trained CNN model, and classifies them using a trained machine learning model. It also provides a **web-based interface** for easy interaction.

---

## 🚀 Features
- 📹 Upload video for analysis  
- 🧠 Deep feature extraction using **VGG16**  
- 📉 Dimensionality reduction using **PCA**  
- 🎯 Classification using **SVM**  
- 📊 Confidence score display  
- 🎥 Output video with result overlay  
- 📂 History of previously analyzed videos  
- 🌐 User-friendly Flask web interface
